* ANSWER to Practice Problem
  * Base case: n = 0
    * action: print(n)
  * First recursive case: n = 1
    * action: print(n)  countdown(n-1)  print(n)
```python
def countdown(n):
  if n = 0:
    print(n)
  else:
    print(n)
    countdown(n-1)
    print(n)
```
